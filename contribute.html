Contribute to Cartilage

We have open sourced Cartilage (under the very liberal MIT license) in the hopes that others may find what we have built useful. There are many ways in which you can contribute to the project:

Contributing Code

Cartilage is hosted on GitHub. If you would like to contribute to the project, please fork and open a pull request with your enhancements!

Report Issues

If through using Cartilage you come across an issue, please take a moment and file an issue with us at GitHub (or fix the issue and open a pull request).
