Getting Started with Cartilage

Requirements

  • Ruby on Rails 3.2+ w/Asset Pipeline
  • Backbone.js 0.9.2+
  • Bootstrap 2.2+
  • Sass
  • CoffeeScript

Installing Cartilage

Cartilage is distributed as a gem file that you may include in your Ruby on Rails project using Bundler:

gem 'cartilage'

You may also pull down the latest development version of Cartilage by pointing to our git repository:

gem 'cartilage', git: 'git://github.com/Cartilage/cartilage.git'

Include Cartilage in the Asset Pipeline

Cartilage is designed to be used with the Asset Pipeline support available in Ruby on Rails. You will need to add Cartilage to your asset manifests (i.e. application.js and application.css):

//= require cartilage
